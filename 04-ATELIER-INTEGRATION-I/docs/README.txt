INTEGRATION D'UNE MAQUETTE GRAPHIQUE HTML / CSS
Voici quelques consignes avant de  commencer :

	1. Vous devez réaliser un rendu fidèle au model fourni ;
	2. Utilisation de CSS / SASS Libre ;
	3. Utilisation de GRID / FLOAT / FLEX au choix ;
	4. Les polices sont Roboto Condensed en 100 | 300 et 700 et Arial ;
	5. Pour les drapeaux de langues, vous devrez utiliser les SPRITES CSS ;
	6. Il se peux que le graphiste est fait des erreurs, à vous d'agir en conséquence.
	7. La réalisation du FOOTER n'est pas OBLIGATOIRE ;
	8. BONUS : Mettre en place un SLIDER à la place de l'image du slider.

Pour vous aider dans vos recherches, voici les notions abordés dans cet Atelier :

	- Positionnement Float
	- Positionnement Relatif / Absolu
	- CSS Sprites
	- CSS Transitions
	- Pseudo Class :before et :after
	- Pseudo Selector :nth-child
	- CSS Linear et Radial Gradient
